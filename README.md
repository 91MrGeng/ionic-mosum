# ionic2-mosum
Ionic 2 Weather App

## Getting Started

* Clone this repository

* Install Ionic, cordova and node_modules

    ```bash
    $ npm install -g ionic
    $ sudo npm install -g cordova@6.3.1
    $ npm install
    ```

### Android

```bash
    $ ionic platform add android
    $ ionic build android
    $ ionic run android
```

### iOS
```bash
    $ ionic platform add ios
    $ ionic build ios
```    
    Run using XCode
    
### icons
Run post_install script
```bash
    $ ./post_install.sh
```
